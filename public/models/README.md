# Modelos 3D Placeholder

Os modelos 3D das moléculas devem ser colocados nesta pasta.

## Como obter modelos GLB reais:

### Opção 1: NIH 3D (Recomendado)
1. Acesse: https://3d.nih.gov/
2. Busque pela molécula (ex: "glucose", "caffeine", "water")
3. Baixe o arquivo GLB
4. Renomeie conforme a lista abaixo e coloque nesta pasta

### Opção 2: PubChem
1. Acesse: https://pubchem.ncbi.nlm.nih.gov/
2. Busque pela molécula
3. Baixe o arquivo SDF
4. Converta para GLB usando ferramentas online ou Blender

## Modelos necessários:

- water.glb - Água (H₂O)
- dna.glb - DNA
- glucose.glb - Glicose (C₆H₁₂O₆)
- caffeine.glb - Cafeína (C₈H₁₀N₄O₂)
- methane.glb - Metano (CH₄)
- hemoglobin.glb - Hemoglobina
- co2.glb - Dióxido de Carbono (CO₂)
- sulfuric-acid.glb - Ácido Sulfúrico (H₂SO₄)
- benzene.glb - Benzeno (C₆H₆)
- aspirin.glb - Aspirina (C₉H₈O₄)
- neuron.glb - Neurônio
- influenza.glb - Vírus da Gripe

## Modelo Placeholder Temporário

Enquanto você não adiciona os modelos reais, a aplicação usará um cubo simples como placeholder.
Para adicionar os modelos, basta fazer o download e colocar nesta pasta com os nomes listados acima.
